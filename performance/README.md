# Mobile Scan Performance Harness

This folder contains the k6 suite and manifest outputs for mobile scan API load testing.

- `k6/mobile_scans.js`: main entry point
- `k6/lib/`: shared auth, payload, classification, config, and summary helpers
- `results/`: default destination for JSON summary output

See [docs/mobile_scan_performance.md](/mnt/c/headless_projects/PETAL_blueprint_project/FastCheckin/docs/mobile_scan_performance.md) for the full runbook.
